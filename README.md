|   OS    |                               PATH                               |
| :-----: | :--------------------------------------------------------------: |
| windows |   HOME\AppData\Local\Google\Chrome\User Data\Default\bookmarks   |
| darwin  | HOME/Library/Application Support/Google/Chrome/Default/Bookmarks |
|  linux  |                HOME/.config/google-chrome/Default                |
